# Clinials - Google Cloud Integration Website

This website showcases Clinials' AI-powered clinical trial document platform and its integration with Google Cloud services.

## Website Structure
- `index.html` - Main website page
- `css/style.css` - All styling
- `js/script.js` - Interactive functionality
- `img/` - Logo and images
- `.github/workflows/` - GitHub Actions for deployment

## Features
- Responsive design for all devices
- Google Cloud integration showcase
- Interactive components
- Smooth scrolling navigation
- Contact form

## Deployment to GitHub Pages

1. Push this code to a GitHub repository
2. Go to Repository Settings > Pages
3. Select "Deploy from a branch" and choose `main` branch
4. Save - Your site will be available at `https://[username].github.io/[repository-name]`
